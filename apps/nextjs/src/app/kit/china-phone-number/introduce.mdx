## 🚀 Features

- Supports generating 1 to 89 phone numbers at a time
- All numbers follow the official China phone number structure
- Responsive design, mobile-friendly

## China Phone Number Format

A China phone number consists of 11 digits and follows the format `xxx-yyyy-zzzz`.

- **xxx** is the network operator, such as [中国移动](https://www.10086.cn), [中国联通](https://www.10010.com), or [中国电信](https://www.189.cn).
- **yyyy** is the area code.
- **zzzz** is the user number.

### Number Segments for `中国移动` (China Mobile)

134, 135, 136, 137, 138, 139, 147, 148, 150, 151, 152, 157, 158, 159, 165, 172, 178, 182, 183, 184, 187, 188, 198

### Number Segments for `中国联通` (China Unicom)

130, 131, 132, 145, 146, 155, 156, 166, 171, 175, 176, 185, 186

### Number Segments for `中国电信` (China Telecom)

133, 149, 153, 173, 174, 177, 180, 181, 189, 191, 199
